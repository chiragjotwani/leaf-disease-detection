# Project Description — Plant Leaf Disease Detection Using Deep Learning

Plant diseases pose a serious threat to agricultural productivity and food security, especially in regions where large-scale manual monitoring is difficult. Traditional disease identification relies heavily on expert visual inspection, which is time-consuming, subjective, and impractical for continuous field surveillance. This project addresses that challenge by developing an automated plant leaf disease detection system using deep learning–based image classification. The objective is to design a reliable computer vision pipeline that can accurately identify leaf diseases from images and compare multiple state-of-the-art neural network architectures to determine the most effective model.

The system is built using a labeled dataset of plant leaf images containing both healthy and diseased samples across multiple categories. The dataset is organized into class-wise directories to support supervised learning. Images exhibit natural variations such as differences in lighting conditions, leaf orientation, scale, and background clutter, making the classification task more realistic and challenging. To ensure balanced evaluation and proper generalization, the dataset is divided into training and validation subsets.

A structured preprocessing pipeline is implemented to prepare the data for model training. All images are resized to a uniform resolution of 224×224 pixels to match the input requirements of standard convolutional neural networks. Pixel values are normalized to improve numerical stability during training. Data augmentation techniques, including random horizontal flipping, rotation, and zoom transformations, are applied to artificially increase dataset diversity and reduce overfitting. The data loading pipeline is optimized using batching, caching, shuffling, and prefetching to ensure efficient utilization of computational resources.

The project leverages transfer learning by utilizing pretrained convolutional neural networks that were originally trained on the ImageNet dataset. Instead of training models from scratch, learned visual features are reused to improve performance and reduce training time. Three widely used architectures are implemented and compared.

The first model is MobileNetV2, a lightweight convolutional neural network designed for efficiency and mobile-friendly applications. It uses depthwise separable convolutions and inverted residual blocks to significantly reduce computational cost while maintaining strong feature extraction capability. This model serves as a fast and resource-efficient baseline for performance comparison.

The second architecture is ResNet50, a deep residual network consisting of fifty layers. It introduces residual connections that allow gradients to flow directly across layers, effectively solving the vanishing gradient problem and enabling deeper network training. Due to its strong representational power, ResNet50 is well-suited for capturing complex visual patterns in diseased leaf textures.

The third model is EfficientNetB0, a modern architecture that applies compound scaling to balance network depth, width, and input resolution. This design achieves a superior tradeoff between accuracy and computational efficiency. EfficientNetB0 is included in the study as an optimized architecture expected to deliver strong performance with fewer parameters.

For each architecture, the original classification layers are replaced with a custom head tailored for the disease classification task. This head includes a data augmentation layer, global average pooling to reduce spatial dimensions, fully connected dense layers for feature learning, dropout layers for regularization, and a softmax output layer for multi-class prediction. This unified design ensures a fair comparison across all models.

Model training is performed using the Adam optimizer and sparse categorical cross-entropy loss function. A consistent set of hyperparameters is maintained across experiments, including image size, batch size, and number of training epochs. Several training callbacks are used to enhance performance, including early stopping to prevent overfitting, learning rate reduction when validation performance plateaus, and model checkpointing to preserve the best-performing weights.

To evaluate performance comprehensively, multiple metrics are used rather than relying solely on accuracy. Precision measures the correctness of disease predictions, recall evaluates the model’s ability to detect actual disease cases, and the F1 score provides a balanced harmonic mean of precision and recall, which is particularly important for imbalanced datasets. These metrics enable a more reliable and fair comparison of classification performance.

Experimental results include validation accuracy tracking across training epochs, quantitative metric comparison among models, and visual performance analysis. A structured comparison framework is used to determine the best-performing architecture based on overall classification effectiveness. The model achieving the highest F1 score is selected as the final optimal model.

This project contributes an end-to-end deep learning framework for plant disease detection, a comparative analysis of prominent transfer learning architectures, and a practical demonstration of artificial intelligence applications in agriculture. The developed system can support smart farming solutions, automated crop monitoring, mobile disease diagnosis tools, and early warning systems, ultimately helping improve crop health management and agricultural productivity.
